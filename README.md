# RAWG (rawg)
RAWG is the largest video game database and game discovery service, providing access to more than 350,000 games across 50 platforms with rich metadata including tags, genres, developers, publishers, creators, release dates, and Metacritic ratings.

## APIs

### RAWG Video Games Database API
The RAWG Video Games Database API provides programmatic access to a catalog of more than 350,000 games across 50+ platforms, including developers, publishers, creators, genres, tags, stores, and platform metadata. API key authentication is required for all requests.

**Human URL:** [https://rawg.io/apidocs](https://rawg.io/apidocs)
